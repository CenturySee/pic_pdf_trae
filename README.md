# 图片PDF转换工具

一个功能强大的在线图片PDF转换工具，支持图片转PDF和PDF转图片双向转换，提供直观的用户界面和丰富的自定义选项。

## 试用地址
[PIC_PDF_Converter](https://tools.imak.top:8443/pic_pdf/)

## ✨ 功能特点

### 📸 图片转PDF
- 支持上传多张图片
- 支持拖拽上传图片
- 支持图片预览和管理
- 支持图片旋转（90度递增）
- 支持图片排序（按文件名或上传时间）
- 支持自定义页面大小（A4、A3、A5、Letter、Legal、自定义）
- 支持页面朝向设置（自动、纵向、横向）
- 支持自定义页边距
- 支持自定义PDF文件名
- 实时进度显示

### 📄 PDF转图片
- 支持上传PDF文件
- 支持拖拽上传PDF
- 支持PDF预览
- 支持调整转换清晰度（DPI：96、150、300、600）
- 支持多种输出格式（PNG、JPG、WebP）
- 实时转换进度显示

## 🚀 快速开始

### 在线使用
1. 直接打开 `index.html` 文件
2. 选择所需的转换模式（图片转PDF或PDF转图片）
3. 上传文件并设置转换选项
4. 点击转换按钮开始转换
5. 下载生成的文件

### 本地部署
1. 克隆或下载项目到本地
2. 使用HTTP服务器运行项目（推荐使用Live Server、http-server等）
3. 在浏览器中访问服务器地址

## 🛠️ 技术栈

- **HTML5** - 页面结构
- **CSS3** - 样式设计
- **JavaScript (ES6+)** - 交互逻辑
- **第三方库**：
  - [pdf-lib](https://pdf-lib.js.org/) - PDF创建和编辑
  - [pdf.js](https://mozilla.github.io/pdf.js/) - PDF渲染和预览
  - [jszip](https://stuk.github.io/jszip/) - 文件压缩
  - [FileSaver.js](https://github.com/eligrey/FileSaver.js/) - 文件下载

## 📖 使用指南

### 图片转PDF
1. 点击"上传图片"按钮或拖拽图片到上传区域
2. 在预览区域可以：
   - 点击旋转按钮旋转图片
   - 点击删除按钮删除图片
   - 拖拽图片调整顺序
3. 设置转换选项：
   - 选择页面大小
   - 选择页面朝向
   - 设置页边距
   - 输入PDF文件名
4. 点击"生成PDF"按钮
5. 等待转换完成后自动下载

### PDF转图片
1. 点击"上传PDF"按钮或拖拽PDF到上传区域
2. 等待PDF加载完成后可以预览
3. 设置转换选项：
   - 选择清晰度（DPI）
   - 选择输出图片格式
4. 点击"转换为图片"按钮
5. 等待转换完成后自动下载ZIP压缩包
